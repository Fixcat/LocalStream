🚀 БЫСТРЫЙ СТАРТ:

1. Установите Node.js (если еще не установлен):
   https://nodejs.org/

2. Откройте командную строку в этой папке

3. Установите зависимости:
   npm install

4. Запустите:
   start.bat

5. Готово! Браузер откроется автоматически

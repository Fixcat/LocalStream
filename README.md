# Приложение для трансляции

Локальное приложение для трансляции камеры и экрана с поддержкой Tailscale.

## Что исправлено:
- ✅ Правильная передача видео через Socket.IO (Blob ↔ ArrayBuffer)
- ✅ Стабильная работа без зависаний
- ✅ Оптимизированная производительность (15 FPS, качество 70%)
- ✅ Поддержка Tailscale для удаленного доступа

## Запуск:

```
start.bat
```

Автоматически откроется страница трансляции.

## Использование:

### Для стримера:
1. Откройте: http://localhost:3000/streamer.html
2. Введите название трансляции
3. Нажмите "Запустить"
4. Разрешите доступ к камере и экрану

### Для зрителя:
1. Откройте: http://localhost:3000/watch.html
2. Выберите трансляцию из списка
3. Переключайтесь между камерой и экраном кнопками

### Через Tailscale:
- Замените `localhost` на Tailscale IP из консоли сервера (100.x.x.x)
- Пример: `http://100.86.241.127:3000/watch.html`

## Технические характеристики:
- **FPS**: 15 (оптимально для стабильности)
- **Качество JPEG**: 70%
- **Разрешение камеры**: 640x480
- **Разрешение экрана**: 1280x720
- **Транспорт**: polling + websocket

## Файлы:
- `server.js` - сервер приложения
- `start.bat` - скрипт запуска
- `public/streamer.html` - страница трансляции
- `public/streamer.js` - код трансляции
- `public/watch.html` - страница просмотра
- `public/watch.js` - код просмотра
- `public/style.css` - стили
- `install_tailscale.bat` - установка Tailscale
- `TAILSCALE_SETUP.md` - инструкция по Tailscale
- `chrome-unsafe.bat` - Запускает Chrome который не считает HTTPS ссылки не безопасными

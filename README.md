# StreamBridge - Мост в мир прямых трансляций

Инновационное приложение для потоковой передачи видео, аудио и экрана в реальном времени с поддержкой удаленного доступа через Tailscale.

## Особенности:
- ✅ Ультра-стабильная передача видео через Socket.IO (Blob ↔ ArrayBuffer)
- ✅ Надежная работа без зависаний и обрывов
- ✅ Оптимизированная производительность (15 FPS, качество 70%)
- ✅ Удаленный доступ через Tailscale для трансляций из любой точки мира
- ✅ Поддержка одновременной трансляции камеры, микрофона и экрана
- ✅ Интуитивный интерфейс для стримеров и зрителей

## Быстрый старт:

```
start.bat
```

Приложение автоматически запустится и откроется главная страница трансляции.

## Использование:

### Для стримера:
1. Откройте: http://localhost:3000/streamer.html
2. Введите уникальное название трансляции
3. Нажмите "Запустить трансляцию"
4. Предоставьте доступ к камере, микрофону и экрану

### Для зрителя:
1. Откройте: http://localhost:3000/watch.html
2. Выберите активную трансляцию из списка
3. Управляйте просмотром: переключайтесь между камерой и экраном, регулируйте звук

### Удаленный доступ через Tailscale:
- Замените `localhost` на ваш Tailscale IP-адрес, отображаемый в консоли (100.x.x.x)
- Пример: `http://100.86.241.127:3000/watch.html`
- Транслируйте из любой точки мира с молниеносной скоростью!

## Технические характеристики:
- **Частота кадров**: 15 FPS (идеальный баланс качества и стабильности)
- **Качество изображения**: JPEG 70% (оптимальное сжатие без потерь качества)
- **Разрешение камеры**: 640x480 (четкое изображение без перегрузки канала)
- **Разрешение экрана**: 1280x720 (HD-качество для комфортного просмотра)
- **Протокол передачи**: polling + websocket (максимальная совместимость и скорость)

## Файлы:

- `public/streamer.html` - страница трансляции

- `public/watch.html` - страница просмотра
- `public/watch.js` - код просмотра
- `public/style.css` - стили
- `install_tailscale.bat` - установка Tailscale
- `TAILSCALE_SETUP.md` - инструкция по Tailscale
- `chrome-unsafe.bat` - Запускает Chrome который не считает HTTPS ссылки не безопасными
